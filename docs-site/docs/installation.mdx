# Installation

There are several ways to install Mystral Native.js depending on your needs.

## Quick Install (Recommended)

<InstallTabs />

This will:
1. Detect your platform and architecture
2. Download the latest release
3. Extract to `~/.mystral/` (macOS/Linux) or `$HOME\.mystral\` (Windows)
4. Add `mystral` to your PATH

## Manual Download

Download the appropriate build for your platform from the [releases page](https://github.com/mystralengine/mystralnative/releases):

| Platform | Recommended Build |
|----------|-------------------|
| macOS (Apple Silicon) | `mystral-macOS-arm64-v8-dawn.zip` |
| macOS (Intel) | `mystral-macOS-x64-v8-dawn.zip` |
| Windows | `mystral-windows-x64-v8-dawn.zip` |
| Linux | `mystral-linux-x64-v8-dawn.zip` |

**Dawn builds** are recommended for best WebGPU shader compatibility. **V8 builds** provide full ES2024+ JavaScript support with JIT compilation.

### After Download

```bash
# macOS/Linux
unzip mystral-macOS-arm64-v8-dawn.zip
cd mystral-macOS-arm64-v8-dawn
./mystral --version

# Windows (PowerShell)
Expand-Archive mystral-windows-x64-v8-dawn.zip
cd mystral-windows-x64-v8-dawn
.\mystral.exe --version
```

## Build from Source

For development or custom configurations:

```bash
# Clone the repository
git clone https://github.com/mystralengine/mystralnative.git
cd mystralnative

# Install bun (if not already installed)
curl -fsSL https://bun.sh/install | bash

# Download dependencies and build
bun install
bun run deps:download
bun run configure
bun run build

# Verify the build
./build/mystral --version
```

See [Building from Source](/mystralnative/docs/guides/building) for advanced build options.
